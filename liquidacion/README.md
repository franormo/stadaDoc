# Proceso de liquidación

El proceso de liquidación automatica toma los ficheros proporcionados por los [mayoristas](/glosario.md#mayorista) y los procesa para estandarizar la información de las ventas de modo que pueda ser tratada de forma homogénea independientemente de su origen. Para lograrlo hace uso de diversas configuraciones en las que se modela cómo la información ha de ser tratada.

En esta sección podemos ver en detalle como se debe configurar el modelo de liquidación y cómo funciona el proceso automátizado.

- [Configuración del sistema](/liquidacion/configuracion.md)
- [Guia de funcionamiento del sistema](/liquidacion/funcionamiento.md)
