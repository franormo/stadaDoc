# Configuración del sistema de liquidaciones

La configuración del sistema permitirá modificar el comportamiento del mismo, definir condiciones comerciales, agrupar [socios](/glosario.md#socio) y todo lo que se necesite para obtener la liquidación de forma automática. Existen varios grupos de datos que se han de configurar, algunos de ellos son básicos para el funcionamiento del sistema, como definir los [mayoristas](/glosario.md#mayorista) que existen, los [clubs](/glosario.md#club-y-subclub), los [materiales](/glosario.md#material) que se venden etc... y otros son mas complejos y detallados, como las reglas que imponen los distintos mayoristas para proporcionar descuentos por volumen de ventas o los costes de gestión asociados a dichos descuentos.
